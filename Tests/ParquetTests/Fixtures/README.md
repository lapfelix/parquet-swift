### Generator Scripts

**Files:**
- `generate_repeated_fixtures.py` - Generates repeated column test fixtures

**Description:** Python scripts that use PyArrow to generate test fixtures with specific data patterns for testing array reconstruction, nullable columns, and other Parquet features.

**Usage:**
```bash
# Generate repeated column fixtures
python3 generate_repeated_fixtures.py
```

## Test Data Descriptions

### Repeated Column Fixtures

Generated by `generate_repeated_fixtures.py` to test array reconstruction:

1. **repeated_int32_simple.parquet** - Basic nested arrays: `[[1, 2], [3], [4, 5, 6]]`
2. **repeated_int32_empty.parquet** - Arrays with empty lists: `[[1, 2], [], [3]]`
3. **repeated_int32_nulls.parquet** - Arrays with null elements: `[[1, None, 2], [None], [3, 4]]`
4. **repeated_int32_all_empty.parquet** - All empty arrays: `[[], [], []]`
5. **repeated_int32_single.parquet** - Single-element arrays: `[[1], [2], [3], [4], [5]]`
6. **repeated_int32_large.parquet** - Large array with 100 elements: `[[1, 2, 3, ..., 100]]`
7. **repeated_int64.parquet** - Int64 arrays: `[[100, 200], [300]]`
8. **repeated_float.parquet** - Float arrays: `[[1.5, 2.5], [], [3.5]]`
9. **repeated_double.parquet** - Double arrays: `[[1.1, 2.2], [3.3, 4.4]]`
10. **repeated_string.parquet** - String arrays: `[["Alice", "Bob"], [], ["Charlie"]]`
11. **repeated_mixed.parquet** - Multiple repeated columns with different types

### Other Test Fixtures

- **pyarrow_test.parquet** - Generated with PyArrow 21.0.0 to test compatibility with Python ecosystem files (pandas, PyArrow, Dask)
- **alltypes_plain.parquet** - Dictionary encoding test file from Apache Parquet project
- **datapage_v1-snappy-compressed-checksum.parquet** - Snappy compression test file from Apache Parquet project
